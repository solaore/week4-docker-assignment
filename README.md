## ✅ How to submit

1. **Fork or clone** this repo (or rebuild the structure in your own `devops-journey` repo under `week4/`).
2. Complete **both** assignments — each in its own folder.
3. Add your screenshots and fill in the report/README where asked.
4. Push to **your own public GitHub repo**.
5. Drop your repo link in **#assignments** on Slack:

```
Week 4 — Docker Assignments
GitHub: https://github.com/YOUR_USERNAME/YOUR_REPO
```

**Deadline:** Before next Saturday's class.
⚠️ Your repo must be **public** — private links will not be marked.

---

## 🐳 Before you start

Make sure Docker is installed and running:
```bash
docker --version
docker compose version
```
Both commands should return a version number. If not, install **Docker Desktop** (Mac/Windows) or **Docker Engine** (Linux) first.
